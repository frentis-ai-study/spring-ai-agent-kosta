[KOSTA 쇼핑몰 주문 정책]

1. 결제 및 주문 확정
- 결제가 완료된 시점부터 주문이 확정됩니다. 결제 전에는 PENDING 상태로 유지되며, 30분 내 결제가 완료되지 않으면 자동 취소됩니다.
- 주문 확정 후 상품 준비 단계로 전환되며, 평일 기준 1영업일 이내 발송을 원칙으로 합니다.

2. 주문 취소
- 상품이 발송되기 전(PENDING, PAID 상태)에는 고객이 직접 취소할 수 있으며, 취소 즉시 결제 수단으로 환불 절차가 시작됩니다.
- SHIPPED 상태 이후에는 직접 취소가 불가하며, 고객센터로 반품 신청을 하여야 합니다.
- DELIVERED(배송 완료) 상태의 주문은 취소 처리할 수 없으며, 반품 정책에 따라 진행됩니다.

3. 배송
- 일반 배송은 결제 완료 후 영업일 기준 1~3일 이내 도착을 목표로 합니다.
- 도서·산간 지역은 추가 1~2일이 소요될 수 있습니다.
- 5만 원 이상 주문 시 무료배송이며, 미만일 경우 3,000원의 배송비가 부과됩니다.

4. 등급별 혜택
- BASIC 회원: 기본 적립 1%
- GOLD 회원: 적립 2% + 익일배송 우선 처리
- VIP 회원: 적립 3% + 무료 반품 + 전용 상담 채널
